## Requirements

- Python 3.7 or higher
- Recommended: Virtual environment for isolation
- Recommended packages (depending on scripts used):
  - requests
  - beautifulsoup4
  - pandas
  - openpyxl
  - psutil

Install required packages:
```bash
pip install -r requirements.txt
